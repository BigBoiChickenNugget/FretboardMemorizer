Instructions

Sooo you wanna memorize the fretboard? Look no further. Just click start game and begin. Some basic information about how it works:
1. Note will flash - A note name will flash on the screen. You will have approximately 3 seconds to start picking that note. Important: You NEED to tremolo pick that note because the backend is kinda funky...
2. Pick note - If your note was correct, good job! If you get 3 correct notes in a row, the duration will decrease by 0.5 seconds. If you get 1 wrong note, the duration will increase by 1 second. The duration does not drop below 1 second.
3. Manual selection - If you know your level and just want to practice, you can set the duration yourself.
